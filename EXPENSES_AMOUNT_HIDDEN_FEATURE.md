# 借贷金额隐藏功能说明

## 功能概述

新增了借贷金额隐藏功能，确保只有通过密码认证的用户才能查看具体的借贷金额，保护个人隐私。

## 功能特性

### 1. 智能金额隐藏
- **未认证状态**：所有金额显示为 "***"
- **已认证状态**：显示真实金额数值
- **动态切换**：认证状态改变时自动更新显示

### 2. 点击认证
- **统计卡片**：点击隐藏的金额触发认证
- **表格记录**：点击隐藏的金额触发认证
- **统一体验**：所有金额元素都支持点击认证

### 3. 视觉反馈
- **锁定图标**：隐藏金额前显示 🔒 图标
- **悬停效果**：鼠标悬停时显示可点击状态
- **认证提示**：点击后弹出密码输入框

## 技术实现

### 1. 认证检查逻辑
```javascript
// 检查用户是否已认证
const isAuthenticated = window.AuthHelper && window.AuthHelper.isAuthenticated(borrower);

// 根据认证状态显示金额
const amountDisplay = isAuthenticated ? `¥${amount}` : '***';
```

### 2. 点击事件处理
```javascript
// 为未认证的金额添加点击事件
if (!isAuthenticated) {
    element.className = 'value amount-hidden';
    element.onclick = () => this.showAmountAuth(borrower);
} else {
    element.className = 'value';
    element.onclick = null;
}
```

### 3. CSS样式设计
```css
.amount-hidden {
    color: #999 !important;
    cursor: pointer;
    transition: all 0.3s ease;
}

.amount-hidden:hover {
    color: #666 !important;
    background: rgba(0, 0, 0, 0.05);
    border-radius: 4px;
    padding: 2px 4px;
}

.amount-hidden::before {
    content: "🔒";
    margin-right: 4px;
    font-size: 0.8em;
    opacity: 0.7;
}
```

## 应用范围

### 1. 统计卡片金额
- **借款总额**：`expenses-hjf-total`, `expenses-hjm-total`
- **已归还金额**：`expenses-hjf-returned`, `expenses-hjm-returned`
- **未归还金额**：`expenses-hjf-pending`, `expenses-hjm-pending`

### 2. 表格记录金额
- **借款记录**：每行记录的金额列
- **还款记录**：每行记录的金额列
- **指定还款信息**：还款记录中的目标金额

### 3. 删除线提示信息
- **归还金额**：提示信息中的金额显示
- **总借款金额**：提示信息中的金额显示

## 用户体验

### 1. 初始状态
- 用户首次访问时，所有金额显示为 "***"
- 金额前显示锁定图标，提示需要认证
- 鼠标悬停时显示可点击状态

### 2. 认证过程
- 点击任意隐藏金额触发认证
- 弹出密码输入框
- 输入正确密码后自动显示金额

### 3. 认证后状态
- 所有相关金额自动显示真实数值
- 移除锁定图标和点击事件
- 保持认证状态直到用户主动清除

## 安全特性

### 1. 权限控制
- 只有通过密码验证的用户才能查看金额
- 支持交叉认证（基于数据量规则）
- 认证状态存储在本地，页面刷新后保持

### 2. 数据保护
- 未认证用户无法看到任何具体金额
- 服务器端不存储敏感金额信息
- 所有金额计算在客户端进行

### 3. 用户体验
- 认证成功后无需重复输入密码
- 支持快速切换认证状态
- 提供清晰的视觉反馈
