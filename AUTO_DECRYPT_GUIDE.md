# 🔓 自动解密功能指南

## 🎯 新功能概述

已成功实现密码自动保存和解密功能，大幅提升用户体验：

### ✅ 核心改进
- **一次输入，永久记住**：首次输入密码后自动保存到localStorage
- **无感解密**：再次访问时自动使用保存的密码解密
- **智能恢复**：密码错误时自动清除并提示重新输入
- **用户控制**：提供"清除已保存密码"选项

## 🔄 使用流程

### 首次访问
1. 打开加密页面
2. 显示密码输入界面
3. 输入正确密码
4. **密码自动保存到localStorage（明文存储）**
5. 页面解密并正常显示

### 后续访问
1. 打开同一加密页面
2. **自动检测存储的密码**
3. **显示加载界面并自动解密**
4. 直接显示解密内容，无需重新输入密码

### 密码管理
- **重置密码**：点击"清除已保存密码"按钮
- **更换密码**：清除后重新输入新密码
- **多设备同步**：密码仅存储在当前浏览器的localStorage中

## 🛡️ 安全说明

### 存储方式
- **位置**：浏览器localStorage
- **格式**：明文存储（便于WASM解密使用）
- **作用域**：仅限当前域名和端口
- **持久性**：除非手动清除或清空浏览器数据

### 安全考虑
- 密码存储在本地，不会发送到服务器
- 仅在相同域名下可访问
- 用户可随时清除存储的密码
- 适合个人设备使用

## 🔧 技术实现

### localStorage存储
```javascript
// 密码保存
localStorage.setItem('wasm_decrypt_password', password);

// 密码读取
const storedPassword = localStorage.getItem('wasm_decrypt_password');

// 密码清除
localStorage.removeItem('wasm_decrypt_password');
```

### 自动解密逻辑
1. **页面加载**：检查localStorage中是否有密码
2. **有密码**：显示加载界面，尝试自动解密
3. **无密码**：显示密码输入界面
4. **解密失败**：清除无效密码，显示输入界面
